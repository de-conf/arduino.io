API Quick Start

All of our examples use the curl command line tool already available on most systems.

The ^^ My IoT Platform ^^ API lets you send/receive information about your current data and do everything else.

Example: Get Your Current Cards

Request
curl http://$IP/api/users/{{current_user.id }}/cards/

Response
{"1":
 {"_links":
  {"owner_url": /api/users/{{current_user.id}}/,
  "self": /api/cards/-your card reference see Cards Manager -},
 "created": Wed, 24 Jul 2019 01:48:49 GMT,
 "dh11_values":7,
 "gaz_values":0,
 "name": arduino,
 "owner": "{{ current_user.full_name() }}",
 "owner_email": "{{ current_user.email }}",
